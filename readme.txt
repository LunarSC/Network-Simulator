This Network Simulator was created by Thomas Pedraza for the Computer Science 567 class. The project is divided into 3 assignments, each building upon
the last.

To run the program, double click "567Assignment1.jar"

To view the final submission, click on the "Assignment 3" Tab at the top of the screen.

You can pause, resume, refresh, and submit with the buttons underneath the assignment tabs at the top of the screen.
